----------------------------------------------------------------------------------------------------------------------------------
AvalonMM_to_hRAM: conversion between Intel Avalon memory-mapped interface and a generic hyperRAM interface
----------------------------------------------------------------------------------------------------------------------------------
The component is separated in two different sub-components:
	# AvalonMM_to_SSRAM: conversion between Intel Avalon memory-mapped interface and SSRAM interface
	# SSRAM_to_hRAM: conversion between SSRAM interface and hyperRAM interface
----------------------------------------------------------------------------------------------------------------------------------
AvalonMM_to_SSRAM_v1 has been completed and succesfully tested
SSRAM_to_hRAM_v1 has not been completed yet
AvalonMM_to_hRAM_v1 has not been completed yet
